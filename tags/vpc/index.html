GCP VPC Network Peering
在 Google Cloud Platform 中，如果要將兩個不同的 project 或是不同 organization 的 Private Network 相連，可以透過 VPC network peering 的方式來達成，讓一些服務可以在 Private 下互通，降低使用外網的風險來增加安全性。
